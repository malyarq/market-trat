# MarketTrat

Локальное расширение для Chrome, Edge и Yandex Browser. Собирает расходы из чеков Ozon и Wildberries, учитывает возвраты и выгружает CSV.

## Установка

1. Откройте `chrome://extensions`.
2. Включите `Режим разработчика`.
3. Нажмите `Загрузить распакованное расширение`.
4. Выберите эту папку.

## Использование

1. Войдите в Ozon и Wildberries.
2. Нажмите иконку `MarketTrat`.
3. Нажмите `Собрать данные`.
4. Нажмите `Скачать CSV`.

Данные никуда не отправляются: всё работает внутри браузера.
