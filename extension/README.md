# MarketTrat

Локальное расширение для Chrome, Edge и Yandex Browser. Собирает расходы из чеков Ozon, Wildberries и Яндекс Маркета, учитывает возвраты и выгружает CSV.

Версия: `0.9.0`.

## Установка

1. Откройте `chrome://extensions`.
2. Включите `Режим разработчика`.
3. Нажмите `Загрузить распакованное расширение`.
4. Выберите эту папку.

## Использование

1. Войдите в Ozon, Wildberries и Яндекс Маркет.
2. Нажмите иконку `MarketTrat`.
3. Выберите нужные маркетплейсы.
4. Нажмите `Собрать данные`.
5. Нажмите `Скачать CSV`.

CSV получает категории по названию товара. Сервисные строки доставки и сборов отбрасываются, где они не относятся к товарам.

Данные никуда не отправляются: всё работает внутри браузера.
Проверка обновлений и словаря категорий обращается только к GitHub и не отправляет данные расходов.
